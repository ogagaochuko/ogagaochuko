# My Portfolio

A personal portfolio website showcasing projects, skills, and contact information.

## Overview

This repository contains a static portfolio website built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main portfolio page
- `style.css` — styling for layout, typography, and responsive design
- `script.js` — interactive behavior and page enhancements
- `assets/` — images, icons, and other media assets

## Features

- Clean, responsive layout for desktop and mobile
- Highlighted project showcase
- Contact information section
- Smooth navigation and interactive UI elements

## How to Run

1. Open `index.html` in your browser.
2. Or use a local web server for development (optional):
   - Python 3: `python -m http.server`
   - Node.js: `npx serve`

## Customize

- Replace content in `index.html` with your personal information.
- Update styles in `style.css` for custom branding.
- Add new JavaScript interactions in `script.js`.
- Add or replace media files inside `assets/`.
